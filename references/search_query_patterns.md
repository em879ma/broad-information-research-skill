# Search Query Patterns

This file defines the search query generation patterns for each research mode.

## General Pattern

For each task, generate:

1. **Broad query**: topic + domain
2. **Narrow query**: topic + specific subtopic / region / time
3. **Official query**: topic + official / site-specific terms
4. **Alternative query**: synonyms, abbreviations, Chinese/English variants
5. **Time-sensitive query**: latest / 2026 / this month / last 30 days

## 1. Academic Research Example

**Research question**: GLP-1 and kidney transplant donor viability

**Queries:**
- `GLP-1 receptor agonist kidney transplant donor viability`
- `diabetes obesity deceased donor kidney discard HbA1c`
- `donor HbA1c graft survival kidney transplant`
- `GLP-1 chronic kidney disease progression diabetes cohort study`

**Search Strategy:**
- Start with broad medical terms
- Add specific conditions (diabetes, obesity)
- Include outcome measures (graft survival, kidney discard)
- Add study design terms (cohort study, randomized trial)
- Filter by year (last 5 years preferred)

**Database-Specific Queries:**

*Google Scholar:*
- `GLP-1 kidney transplant donor`
- `GLP-1 receptor agonist deceased donor`

*PubMed:*
- `GLP-1[Title/Abstract] AND kidney transplant[Title/Abstract]`
- `diabetes mellitus AND organ donation AND outcomes`

*arXiv (for preprints):*
- `GLP-1 organ transplantation`
- `insulin therapy kidney donation`

## Query Optimization Tips

1. **Use quotes** for exact phrases: `"AI meeting assistant"`
2. **Use site:** to search specific websites: `site:openai.com`
3. **Use intitle:** to find pages with specific words in title: `intitle:AI agent`
4. **Use filetype:** to find specific file types: `AI regulation filetype:pdf`
5. **Use time filters** when available: `AI agents 2026`
6. **Mix Chinese and English** for Chinese topics: `AI µ┤╗ÕŖ© õĖŖµĄĘ`
7. **Add synonyms** to capture more results: `AI agent OR AI assistant OR LLM agent`
